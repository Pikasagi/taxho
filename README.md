# TaxHo - HK Expense Tracker for freelancers and self-employed users

A simple iOS app for Hong Kong freelancers and self-employed users to track income and expenses, keep receipt records, and stay organized for tax season.

This repository hosts public documentation and privacy policy for TaxHo. Source code is maintained in a separate private repository.

## Features

- Quick expense and income entry
- Receipt/invoice scanning and attachment
- Tax-deductible marking and proof reminders
- Daily ledger and fiscal year summary
- Category breakdown and drill-down views
- Export and import (CSV / ZIP bundle)
- Home Screen widget for monthly expense snapshot
- Multi-language support (English, Traditional Chinese, Simplified Chinese)

## Built With

- SwiftUI
- SwiftData
- WidgetKit

## Requirements

- iOS 18.0+
- Xcode 15+
